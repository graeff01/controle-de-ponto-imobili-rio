# Sistema de Ponto - Backend

Backend do sistema de controle de jornada.

## Setup Local

1. Instalar dependências:
```
npm install
```

2. Configurar .env:
```
cp .env.example .env
```

3. Criar banco de dados PostgreSQL

4. Rodar migrations:
```
npm run migrate
```

5. Rodar seeds (dados iniciais):
```
npm run seed
```

6. Iniciar servidor:
```
npm run dev
```

Servidor rodando em: http://localhost:5000
